Los API son kioskos de autoservicio, ubicados en lugares de conveniencia y alto tránsito, con facilidades de parqueo, seguridad y horarios extendidos.

En primera instancia los API estarán ubicados en algunas tiendas de la cadena Ekono y Circle K, quienes son los primeros aliados de Correos en este proyecto.

Los 23 API forman parte de un proceso de expansión del servicio de entrega de paquetería que contempla la apertura de 83 unidades durante el año 2018.
